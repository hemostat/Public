Linear regression model

\(y = X \beta + e \mbox{, where } e \sim (0, \sigma_e^2 R)\)

  • \(y\): vector of \(n\) observations
  • \(X\): the design or incidence \(n \times p\) matrix
  • \(e\): vector of a residual error with covariance matrix \(\sigma_e^2 R\),
    • off-diagonal elements of \(R\) indicate the lack of independence
    • the diagonal elements indicate the heteroscedasticity

http://variani.github.io/talks/2014/01-mixed-models-qtl/#8

Heteroscedastic models

en.wikipedia.org/wiki/Linear_regression

Various models have been created that allow for heteroscedasticity, i.e. the errors for different response variables may have different variances.

Conclusions

  1. Heteroscedasticity is a well known issue in linear models
    • Do some exploratory analysis like boxplots
  2. Unbalanced heteroscedastic groups cause more problems
  3. Mixed model estimates variance components with errors
