What is Common PCA used for?

Instrumental drift (batch) correction in LC/MS metabolomics data (Fernández-Albert et al. 2014)


Ideas why that works

  1. CPCA fits well to the definition of drift (instrumental) noise
  2. The first component is often enough

Common PCA: # LMM effects = 1

Model with a single random effect:

\(y = X \beta + g + e\)

\(\mbox{where } g \sim (0, \sigma_g^2 K) \mbox{ and } e \sim (0, \sigma_e^2 I)\)

\(\mbox{implying y } \sim (X \beta, V) = (X \beta, \sigma_g^2 K + \sigma_e^2 I)\)

The EVD trick (something standard for LMM in GWAS):

\(\sigma_g^2 K + \sigma_e^2 I = \sigma_g^2 U^T D U + \sigma_e^2 U^T I U = U^T [\sigma_g^2 D + \sigma_e^2 I] U\)

Speed-up on a tiny GWAS on the A. thaliana data set, N = 166 species, M = 20 SNPs:

  • LMM: 23.7s
  • EVD + LMM: 1.5s

Common PCA: # LMM effects > 1

Model with many random effects:

\(y = X \beta + \sum_i{g_i} + e\)

\(\mbox{where } g_i \sim (0, \sigma_{i_g}^2 K_i) \mbox{ and } e \sim (0, \sigma_e^2 I)\)

\(\mbox{implying y } \sim (X \beta, V) = (X \beta, \sum_i{\sigma_{i_g}^2 K_i} + \sigma_e^2 I)\)


The Common PCA hypothesis:

\(H_{CPCA}: K_i = U^T D_i U\)


If CPCA exists (the ideal case):

\(\sum_i{\sigma_g^2 K_i} + \sigma_e^2 I = U^T [\sum_i{\sigma_g^2 D_i} + \sigma_e^2 I] U\)

Common PCA: # LMM effects > 1

What are the use cases?

  • Additive genetic + Dominance genetic (hardly the case in human genetics)
  • Additive genetic + Family (pedigrees, especially animals like baboons)
  • Additive genetic + Ancestry (Yu et al. 2006)


Is CPCA computable?

  • The exact CPCA solution exists only if matrices \(K_i\) are commuting (AB = BA)
  • Needed to explore particular cases when CPCA works
    • low-rank of the second (family or ancestry) matrix

Heritability in LMM

Defition 1 in LMM (Bulik-Sullivan et al. 2015):

\(y = X \beta + e\)

\(\beta = argmax_{\alpha} Cor[y, X \alpha]\)

\(h^2_{snp} = \sum{\beta_j^2}\)


Defition 2 in LMM:

\(h^2 = \frac{\sigma^2_g}{\sigma^2_{total}}\)

Heritability on observed and liability scales

On the observed scale:

\(y = \mu 1 + u + e\)

On the liability scale:

\(l = \mu 1 + g + e\)

Assumed as in (Dempster and Lerner 1950):

\(u = Const + bg = Const + cov(y, l) g = Const + zg\)

Result for both both REML (Lee et al. 2011) and LDscore reg. (Bulik-Sullivan et al. 2015) :

\(h^2_{obs_{cc}} = h^2_l \frac{z^2 P (1-P)}{K^2 (1 - K)^2}\)

Heritability in GLMM

In animal breeding studies (Vazquez et al. 2009), (Koeck et al. 2010), a heritability approximation on the \(\eta\)-scale

  1. Variance of a probit random variable: \(1\)

  2. Variance of a logistic random variable: \(\pi^2/3\)
